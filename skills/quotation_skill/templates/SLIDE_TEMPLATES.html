# quotation_skill HTML Templates (Overflow-Proof) ## Overview These HTML templates are **pre-configured with correct margins, fonts, and spacing** to prevent text overflow. Use these as starting points for any proposal template. ## Template Usage 1. **Copy the appropriate template** for your slide type 2. **Replace placeholder content** with your actual content 3. **Adjust title text** in the `

` element 4. **Save as slideXX_name.html** --- ## Template 1: Cover Slide ```html

YOUR_PROPOSAL_TITLE

for CLIENT_NAME

DATE_HERE

``` --- ## Template 2: Content Slide (Bullet Points) ```html

YOUR_SLIDE_TITLE

  • Label: Your content here
  • Label: More content
  • Regular bullet point without label
``` --- ## Template 3: Two-Column Slide ```html

YOUR_SLIDE_TITLE

Left Column Title

  • Item 1
  • Item 2
  • Item 3

Right Column Title

  • Item 1
  • Item 2
  • Item 3
``` --- ## Template 4: Content Slide with Sections ```html

Section 1 Title

  • Item 3
``` --- ## Template 5: Module Slide ```html

/* right=120pt, bottom=85pt */ } /* For slides with MORE content, increase bottom margin */ .content-heavy { margin: 0 140pt 110pt 40pt; /* or 95pt, 100pt if needed */ } /* TYPOGRAPHY */ font-size: 10pt; /* Body text - DO NOT increase */ font-size: 14pt; /* Section headers - max */ font-size: 28pt; /* Slide title */ line-height: 1.15; /* Tight spacing - DO NOT increase */ /* SPACING */ margin: 0 0 8pt 0; /* Paragraph spacing */ margin: 12pt 0 8pt 0; /* Header spacing */ margin-bottom: 2pt; /* List item spacing */ ``` --- ## Quick Reference: Which Template to Use | Content Type | Use Template | Notes | |--------------|-------------|-------| | Cover page | Template 1 | Title + client + date | | Bullet lists | Template 2 | Single column with labels | | Two columns | Template 3 | Side-by-side content | | Sections + bullets | Template 4 | Multiple h2 sections | | AI module | Template 5 | Purpose, alert, preconditions | | Timeline/phases | Template 6 | Multiple phases with tasks | --- ## Content Length Guidelines To prevent overflow, **keep text concise**: - **Purpose**: 1 sentence max - **Descriptions**: 10-15 words per bullet - **Lists**: 5-7 bullets per section - **URLs**: Use short URLs (bit.ly) or separate lines **If content overflows**: 1. **First**: Increase bottom margin by 5pt (85pt → 90pt → 95pt) 2. **Second**: Reduce content length (shorten sentences) 3. **Third**: Reduce font size to 10pt (last resort) 4. **Last**: Split into 2 slides --- ## Background Image Path **IMPORTANT**: Update the background path for your environment: ```bash # Get absolute path echo "file://$(realpath ~/.claude/skills/quotation_skill/assets/background.png)" # Replace in template: background-image: url('file:///YOUR_ABSOLUTE_PATH_HERE'); ``` --- ## Validation After creating slides, **validate** before final output: ```bash cd ~/.claude/skills/pptx node your_convert_script.js ``` **If you get overflow errors**: 1. Check bottom margin is ≥85pt 2. Check font size is ≤11pt 3. Check line-height is ≤1.25 4. Shorten content if needed --- ## Template 7: Two-Column Timeline (For Content-Heavy Slides) **USE THIS** when timeline has many phases/tasks that don't fit in single column. ```html
